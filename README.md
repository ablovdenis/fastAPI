Запуск производить так:
1. Клонируйте репозиторий.
2. В папке проекта (в той папке, внутри которой при использовании команды "ls"/"dir" отображались src/, main.py и прочие) создавайте окружение (версия Python должна быть 3.14.3).
3. Далее через "pip install -r requirements.txt" или "uv pip install -r requirements.txt" импортируете библиотеки, нужные для запуска проекта.
4. Запускаете проект с помощью команды "uvicorn main:app" или "uvicorn main:app --reload".
