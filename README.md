Запуск производить так:
1. Клонируйте репозиторий.
2. В папке проекта (в той папке, внутри которой при использовании команды "ls"/"dir" отображались src/, main.py и прочие) создавайте и запускайте докер-контейнеры и тома командой "docker compose up -d" (версия Python должна быть 3.14.3).
3. Заходите на сервер по адресу "http://localhost:8020/docs#/".
4. Чтобы удалить контейнеры, прописывайте "docker compose down". Дополнительный тег "-v" ликвидирует используемый том (в нём хранятся данные БД).

При выполнении данного задания никто не пострадал (только мои наушники).
